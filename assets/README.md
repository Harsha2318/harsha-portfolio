## üí° Quick Tips

1. **Always compress images** before adding to this folder
2. **Use exact file names** as shown above (or update HTML to match)
3. **Keep file sizes small** for faster loading
4. **Use JPG for photos**, PNG for graphics with transparency
5. **Test images** by opening index.html in browser
